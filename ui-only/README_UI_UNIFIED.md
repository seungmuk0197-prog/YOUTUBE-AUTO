# UI 통일 안내

**일상 개발·운영에서 사용하는 UI는 1개입니다.**

| 용도       | 사용할 앱   | 접속 주소                    | 실행 방법              |
|------------|-------------|-----------------------------|------------------------|
| 프로젝트 목록/편집 | **web-ui** | **http://localhost:3001**   | `start_dev.bat` 또는 `cd web-ui && npm run dev` |

- `ui-only` 폴더는 별도/레거시 목적용입니다.  
- 수정·개발 시 **web-ui** 쪽만 편집하면 됩니다. (두 개를 같이 쓰면 화면·포트가 엉깁니다.)
